# AI Onboarding Copilot — Контекст проекта

## Проблема

Новый сотрудник колл-центра садится за Bitrix24. Перед ним 50+ кнопок, сложные меню, воронки сделок, телефония. Он не знает где что находится. Тратит 2 недели на обучение, постоянно отвлекает коллег с вопросами. Видео-инструкции никто не смотрит. Чат-боты далеко.

## Решение

Chrome-расширение — AI-ассистент который живёт в боковой панели браузера и видит ту же страницу что и сотрудник. Как личный тренер прямо на экране.

## Что умеет

### 1. Авто-анализ экрана
Расширение автоматически анализирует текущую страницу и показывает:
- Название экрана ("Список сделок", "Карточка контакта")
- Контекстную подсказку что делать
- Следующий рекомендуемый шаг

### 2. Пошаговый гид с AI-курсором (главная фича)
Сотрудник вводит вопрос: *"Как создать сделку?"* или *"Как позвонить клиенту?"*

ИИ возвращает пошаговый план. По каждому шагу:
- Делается скриншот страницы
- Отправляется в GPT-4o vision
- GPT-4o видит картинку и возвращает точные пиксельные координаты элемента
- Анимированный AI-курсор (индиговый указатель) летит по параболической дуге точно к нужной кнопке
- Появляется подсказка с инструкцией

Это как Clicky (heyclicky.com) но в браузере.

### 3. Обучение элементам (запись селекторов)
Если GPT-4o промахивается — можно записать элемент вручную:
- Вводишь название: "Создать сделку"
- Нажимаешь "Записать"
- На странице появляется жёлтый баннер
- Кликаешь на нужную кнопку
- Расширение запоминает точный CSS-селектор
- При следующем запросе гид использует его в первую очередь

### 4. База знаний (RAG)
Загружаешь PDF/DOCX инструкции компании. Расширение разбивает их на чанки, создаёт embeddings, хранит в IndexedDB. При запросах ищет релевантные фрагменты и добавляет их в контекст.

### 5. Встроенная карта Bitrix24
60+ элементов с реальными CSS-классами из open-source исходников Bitrix24:
- Канбан сделок (`.crm-kanban-item`, `.main-kanban-column`)
- Телефония (`.im-phone-call-btn-green/red/hold/mute/transfer`)
- Карточки записей, фильтры, тулбар

## Технический стек

| Слой | Технология |
|---|---|
| Расширение | Chrome MV3 |
| UI | React 18 + TypeScript + TailwindCSS |
| Сборка | Vite 6 + vite-plugin-web-extension |
| Состояние | Zustand |
| AI | OpenAI API (gpt-4o-mini для анализа, gpt-4o для vision, text-embedding-3-small) |
| Vector store | IndexedDB (idb) — всё в браузере, без сервера |
| PDF/DOCX | pdfjs-dist + mammoth |

## Архитектура (3 слоя Chrome MV3)

```
Content Script (каждая страница)
  → извлекает DOM-контекст
  → режим записи элементов (желтый баннер + клик)
  → рисует AI-курсор и анимации на странице
  → принимает координаты от background, запускает полёт курсора

Background Service Worker
  → хранит lastContext (переживает засыпание через chrome.storage.session)
  → делает скриншот через chrome.tabs.captureVisibleTab
  → вызывает GPT-4o vision для точных координат
  → оркестрирует все API вызовы
  → роутит сообщения между content и sidepanel

Side Panel (боковая панель React)
  → GuidePanel: ввод запроса, список шагов, прогресс-бар, быстрые подсказки
  → ElementCapturePanel: обучение элементам, подсказки для Bitrix24
  → KBUploader: загрузка документов, прогресс embedding
  → AutoAnalysis: CurrentScreen + AIHint + NextStep + KBInstruction
```

## Как работает точный курсор

1. Пользователь нажимает шаг в гиде
2. CSS-подсветка показывается мгновенно (fallback)
3. Background делает `chrome.tabs.captureVisibleTab` — скриншот
4. Скриншот ресайзится до 1280×800, отправляется в gpt-4o с вопросом
5. gpt-4o возвращает `{"x": 342, "y": 187}` — пиксельные координаты
6. Координаты передаются в content script
7. AI-курсор летит по параболической дуге Безье (как Clicky):
   - Поворачивается по направлению движения (tangent к кривой)
   - Увеличивается на 30% в середине полёта (scale pulse)
   - Приземляется с idle-bounce анимацией
8. Вокруг элемента пульсирует кольцо
9. Текст инструкции набирается посимвольно в bubble

### Запуск для разработчика
```bash
git clone https://github.com/Algorithm-Azamat/call_center_assist.git
cd call_center_assist/extension
npm install
npm run build
# chrome://extensions/ → Режим разработчика → Загрузить распакованное → dist/
```

### Нужен OpenAI API ключ
- Получить: https://platform.openai.com/api-keys
- Вставить в Settings расширения
