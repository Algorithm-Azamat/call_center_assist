# AI Onboarding Copilot

Chrome-расширение — ИИ-помощник для сотрудников колл-центров и корпоративных систем (Bitrix24, CRM, Helpdesk). Анализирует текущую страницу и показывает пошаговый гид с анимированным курсором прямо на экране.

## Возможности

- **Автоанализ экрана** — ИИ определяет что за страница открыта и даёт подсказку
- **Пошаговый гид** — задаёшь вопрос ("Как создать сделку?"), ИИ показывает шаги с анимированным курсором на элементах
- **Обучение элементам** — кликаешь на кнопку → расширение запоминает её селектор, гид становится точнее
- **База знаний** — загружаешь PDF/DOCX инструкции, ИИ использует их при ответах
- **Встроенная карта Bitrix24** — 60+ элементов интерфейса уже записаны

## Быстрый старт

### 1. Клонируй репозиторий

```bash
git clone https://github.com/Algorithm-Azamat/call_center_assist.git
cd call_center_assist/extension
```

### 2. Установи зависимости и собери

```bash
npm install
npm run build
```

После сборки появится папка `extension/dist/`.

### 3. Загрузи в Chrome

1. Открой `chrome://extensions/`
2. Включи **Режим разработчика** (переключатель вверху справа)
3. Нажми **Загрузить распакованное расширение**
4. Выбери папку `extension/dist/`

### 4. Настрой API ключ

1. Нажми `Ctrl+Shift+Y` или кликни иконку расширения
2. В боковой панели нажми шестерёнку (Настройки)
3. Введи OpenAI API ключ → Сохранить

Получить ключ: https://platform.openai.com/api-keys

## Требования

- Node.js 18+
- Chrome 114+
- OpenAI API ключ (поддерживает также Azure OpenAI и Ollama)
